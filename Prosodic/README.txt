Before use: 
1. install espeak
2. install prosodic

Scripts: 
Prosodic_implementation: for processing data. 
SplitFile: for splitting data into 1k line smaller files. 
Prosodic_output_to_syllables: convert prosodic output files' content to strings containing syllables (in the form of "[syl][stress]", e.g., "the0" or "LO1").
